# 字数检查与自动扩写

## 功能说明

写作完成后自动检查字数，不达标时进行精细扩写，确保符合字数范围要求。

## 工作流程

### 1. 自动触发

每次 `/nw write` 完成后，自动执行字数检查：

```
/nw write 写第一章
    ↓ AI生成内容
章节正文（约1800字）
    ↓ 自动检查
字数：1800字 < 最低2000字 ⚠️
    ↓ 自动扩写
精细扩写至2000-4000字范围
    ↓ 完成
章节正文（约2300字）✅
```

### 2. 手动触发

```
/nw check wordcount     # 检查当前章字数
/nw expand 当前章       # 手动扩写当前章
```

## 字数范围配置

在创作宪法中配置：

```markdown
## 字数要求
- 最低字数：2000字（可调整）
- 最高字数：4000字（可调整）
- 理想字数：2500-3500字（推荐）
```

**初始化时可设置**：
```
/nw init 写一本小说，每章2500-3500字
```

## 扩写规则

### 扩写原则

1. **不改变原有情节** - 扩写不改变故事走向
2. **不添加新设定** - 不引入新角色、新物品、新地点
3. **细化描写** - 增加细节、心理、环境描写
4. **丰富对话** - 扩展对话内容、增加角色反应
5. **保持文风** - 使用当前文风扩写

### 扩写优先级

| 优先级 | 扩写方式 | 适用场景 |
|--------|---------|---------|
| 1 | 增加心理描写 | 角色决策、情感变化 |
| 2 | 丰富环境描写 | 场景过渡、氛围营造 |
| 3 | 扩展对话 | 角色互动、信息传递 |
| 4 | 增加动作细节 | 战斗、修炼、操作 |
| 5 | 补充感官描写 | 视觉、听觉、嗅觉、触觉 |

### 扩写示例

**原文（180字）**：
```
叶凡走进山洞，发现里面很黑。他拿出火把照亮，看到墙上有一些壁画。
壁画上画着一些奇怪的图案，像是某种功法。他走近仔细看了看，发现是一门修炼功法。
```

**扩写后（350字）**：
```
叶凡小心翼翼地踏进山洞，脚下的碎石发出细碎的声响。洞内一片漆黑，伸手不见五指，
阴冷的风从深处吹来，带着潮湿的土腥味。

他摸出火折子，吹亮后举在身前。跳动的火光将洞壁照得忽明忽暗，也照亮了墙上那些
斑驳的壁画。

壁画显然年代久远，颜料已经剥落了大半，但仍能辨认出上面绘制的内容。那是一些人形
图案，摆出各种奇怪的姿势，四肢和躯干的线条扭曲得近乎诡异。更引人注目的是，每个
人形周围都环绕着细密的符文，像是某种注释或口诀。

叶凡屏住呼吸，凑近墙面仔细辨认。这些符文他从未见过，但隐隐觉得与修炼有关。他伸出
手指，沿着符文的笔划虚虚描摹，脑海中突然闪过一丝明悟——这似乎是一门失传已久的
修炼功法。
```

**扩写分析**：
- 增加环境细节：碎石声响、阴冷风、土腥味
- 增加动作细节：摸出火折子、吹亮、举起
- 增加感官描写：视觉（火光忽明忽暗）、嗅觉（土腥味）、触觉（阴冷风）
- 增加心理描写：屏住呼吸、隐隐觉得、闪过明悟
- 原文字数：180字 → 扩写后：350字

## 扩写指令

| 指令 | 说明 | 示例 |
|------|------|------|
| `/nw check wordcount` | 检查字数 | `/nw check wordcount` |
| `/nw expand` | 扩写当前章 | `/nw expand` |
| `/nw expand` | 指定扩写量 | `/nw expand 增加500字` |
| `/nw expand` | 定向扩写 | `/nw expand 扩写战斗场景` |

## 字数检查输出

```markdown
## 📊 字数检查报告

### 第X章 [章节标题]
- 当前字数：1,850字
- 目标范围：2,000-4,000字
- 状态：⚠️ 偏低（差150字）

### 建议操作
1. 自动扩写：`/nw expand`
2. 手动扩写：`/nw expand 增加200字`
3. 定向扩写：`/nw expand 扩写对话部分`

### 扩写建议位置
- 第5段：可增加环境描写（+50字）
- 第12段：可扩展对话（+100字）
- 第18段：可增加心理描写（+50字）
```

## 注意事项

### 不要这样做

- ❌ 添加新角色
- ❌ 引入新物品
- ❌ 改变情节走向
- ❌ 重复已有内容
- ❌ 使用水字数技巧

### 应该这样做

- ✅ 细化场景描写
- ✅ 增加角色心理活动
- ✅ 丰富对话内容
- ✅ 补充感官细节
- ✅ 扩展动作描写
