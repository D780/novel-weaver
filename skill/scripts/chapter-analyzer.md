# 章节分析工具

## 功能说明

对章节进行全面分析，包括字数、节奏、爽点、Hook、AI味等多维度分析。

## 使用方法

### 分析当前章节

```
/nw review 当前章节
```

### 分析指定章节

```
/nw review 第三章
```

## 分析维度

### 1. 基础统计

| 指标 | 说明 |
|------|------|
| 总字数 | 章节总字数 |
| 段落数 | 段落数量 |
| 对话行数 | 对话行数 |
| 平均段落长度 | 每段平均字数 |

### 2. 内容结构

| 指标 | 说明 |
|------|------|
| 开场类型 | 场景描写/对话/动作/悬念 |
| 结尾类型 | 悬念/转折/情感/日常 |
| Hook质量 | A/B/C/D 四级评分 |
| 爽点数量 | 本章爽点个数 |
| 伏笔埋设 | 新埋设伏笔数 |
| 伏笔回收 | 回收伏笔数 |

### 3. 节奏分析

| 指标 | 说明 |
|------|------|
| 章节节奏评级 | S1-S5 五级 |
| 节奏变化 | 平缓/上升/高潮/回落 |
| 与前章对比 | 节奏是否变化 |

### 4. 角色分析

| 指标 | 说明 |
|------|------|
| 出场角色 | 本章出场角色列表 |
| 主角戏份占比 | 主角出场时间比例 |
| 对话分布 | 各角色对话量 |
| OOC检测 | 是否有角色崩人设 |

### 5. AI味检测

| 指标 | 说明 |
|------|------|
| 套话数量 | AI高频套话出现次数 |
| 情感空洞 | 是否只有情感告知无描写 |
| 句式重复 | 是否有连续相同句式 |
| AI味评级 | 无/轻微/中等/严重 |

### 6. 质量评分

| 维度 | 权重 | 评分 |
|------|------|------|
| 情节推进 | 25% | 1-10分 |
| 角色塑造 | 20% | 1-10分 |
| 对话质量 | 15% | 1-10分 |
| 场景描写 | 15% | 1-10分 |
| 节奏控制 | 15% | 1-10分 |
| Hook质量 | 10% | 1-10分 |
| **综合评分** | **100%** | **加权平均** |
