# 字数统计脚本

## 功能说明

统计章节字数，检查是否符合目标字数范围。

## 使用方法

### 统计当前章节

```
/nw stats
```

### 统计指定章节

```
/nw stats 第三章
```

### 统计整卷

```
/nw stats volume
```

### 统计全书

```
/nw stats all
```

## 统计规则

### 中文字数计算

- 一个汉字 = 1字
- 一个英文字母 = 1字
- 一个数字 = 1字
- 标点符号不计入字数
- 空行不计入字数

### 字数范围检查

从创作宪法中读取字数范围配置：
- **最低字数**：`constitution.md` 中的 `min_words_per_chapter`（默认 2000）
- **最高字数**：`constitution.md` 中的 `max_words_per_chapter`（默认 4000）

## 输出格式

```markdown
## 📊 字数统计报告

### 当前章节：第X章 [章节标题]
- 实际字数：2850字
- 目标范围：2000-4000字
- 状态：✅ 达标

### 本章各段字数
| 段落 | 字数 | 占比 |
|------|------|------|
| 第1段 | 320字 | 11.2% |
| 第2段 | 450字 | 15.8% |
| ... | ... | ... |

### 对话/描写比例
- 对话：1200字（42.1%）
- 描写：980字（34.4%）
- 心理：420字（14.7%）
- 叙述：250字（8.8%）
```
