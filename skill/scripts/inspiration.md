# 灵感助手

## 功能说明

当作者没有灵感时，提供剧情走向建议、下一步情节推荐、冲突设计等。

## 使用方法

### 基础用法

```
/nw inspire
```

AI会分析当前剧情，给出下一步建议。

### 定向灵感

```
/nw inspire 下一步剧情
/nw inspire 新角色建议
/nw inspire 冲突设计
/nw inspire 爽点建议
/nw inspire 伏笔回收
```

### 描述性灵感

```
/nw inspire 我想写一个主角被困在秘境的情节，有什么好点子？
/nw inspire 最近剧情太平淡了，怎么制造冲突？
/nw inspire 反派太弱了，怎么增强压迫感？
```

## 灵感生成规则

### 生成依据

灵感生成必须基于：
1. **当前剧情进展** - 不脱离已有剧情
2. **角色档案** - 符合角色性格和能力
3. **世界观设定** - 不违反已有设定
4. **伏笔状态** - 优先推进未回收伏笔
5. **大纲规划** - 符合整体故事走向

### 禁忌

- ❌ 引入与现有设定矛盾的内容
- ❌ 角色行为OOC
- ❌ 机械降神式解决方案
- ❌ 与前文矛盾的建议
