# Pacing Analysis 节奏分析

## 系统原理

Pacing Analysis 自动分析章节和卷的节奏，确保张弛有度，
避免连续紧张或连续平淡，维持读者的追读兴趣。

---

## 一、节奏层级

### 1. 章节内节奏（Scene Level）

每个场景的节奏评级：

| 等级 | 名称 | 特征 | 适用场景 |
|------|------|------|---------|
| S5 | 极高潮 | 生死关头、终极对决 | 卷末、大高潮 |
| S4 | 高潮 | 战斗、突破、打脸 | 每10章一次 |
| S3 | 上升 | 准备、计划、推进 | 高潮前铺垫 |
| S2 | 平缓 | 日常、对话、探索 | 高潮后缓冲 |
| S1 | 低谷 | 纯说明、设定介绍 | 开篇、过渡 |

### 2. 章节间节奏（Chapter Level）

连续章节的节奏变化：

```
理想节奏曲线：
S2 → S3 → S4 → S2 → S1 → S3 → S4 → S5 → S2 → S3...
  ^       ^         ^           ^
  铺垫    上升      小高潮      大高潮
```

**节奏规则**：
- 不能连续3章S4或S5（读者疲劳）
- 不能连续3章S1或S2（读者弃读）
- 每5-8章必须有S4或S5
- 每3章必须有S3或更高

### 3. 卷节奏（Volume Level）

整卷的节奏分布：

| 阶段 | 章节范围 | 节奏 | 内容 |
|------|---------|------|------|
| 起 | 1-5章 | S1→S2→S3 | 引入新事件/场景 |
| 承 | 6-15章 | S2→S3→S4 | 发展冲突 |
| 转 | 16-25章 | S3→S4→S4 | 中点转折 |
| 合 | 26-35章 | S4→S5→S2 | 高潮+余韵 |

## 二、节奏分析指标

### 1. Hook 点密度

每章应该有的 Hook 点数量：

| 章节位置 | Hook 点数量 | 类型 |
|---------|------------|------|
| 开篇章 | 2-3 | 引入悬念 |
| 发展章 | 1-2 | 推进悬念 |
| 高潮章 | 1-2 | 结果悬念 |
| 过渡章 | 1-2 | 新事件引子 |

**Hook 冷却**：
- 同类 Hook 不能连续使用
- 每5章必须有 Hook 兑现
- 未兑现的 Hook 不能超过10章

### 2. 爽点密度

| 题材 | 爽点间隔 | 爽点类型要求 |
|------|---------|-------------|
| 爽文 | 每2-3章 | 打脸、逆袭、突破 |
| 悬疑 | 每5-8章 | 解谜、反转 |
| 言情 | 每3-5章 | 感情推进、误会解除 |
| 凡人流 | 每5-10章 | 突破、收获 |

### 3. 信息密度

每章应该包含的信息量：

| 信息类型 | 每章数量 | 说明 |
|---------|---------|------|
| 新设定 | 0-2 | 不能过多新设定 |
| 新角色 | 0-2 | 不能同时引入太多角色 |
| 情节进展 | 1-2 | 必须有实际进展 |
| 伏笔 | 1-3 | 埋设或回收 |

### 4. 情感波动

连续章节的情感变化：

```
理想情感曲线：
轻松 → 紧张 → 感动 → 愤怒 → 爽快 → 温暖 → 悬念...
```

**规则**：
- 不能连续3章同种情感
- 必须有情感对比（先抑后扬）
- 高潮必须有情感爆发

## 三、节奏分析工具

### 章节节奏卡片

```
# 第X章 节奏分析

## 基本信息
- 字数：2500
- 场景数：3
- 节奏等级：S4（高潮）

## 场景节奏
| 场景 | 节奏 | 类型 | Hook | 爽点 |
|------|------|------|------|------|
| 1 | S3 | 准备 | 悬念引子 | - |
| 2 | S4 | 战斗 | - | 打脸反派 |
| 3 | S2 | 收尾 | 新危机 | 获得宝物 |

## 节奏评估
- 本章节奏：✅ 良好（张弛有度）
- Hook数量：2个 ✅
- 爽点数量：1个 ⚠️ 偏少
- 信息密度：适中 ✅
- 情感波动：紧张→爽快 → ✅

## 前后章节对比
- 前3章节奏：S2 → S3 → S4 ✅ 上升
- 后3章建议：S2 → S1 → S3  缓冲后上升

## 问题
- 连续2章战斗场景，建议下一章日常缓冲
- Hook点H005已埋设8章未回收，建议尽快
```

### 卷节奏概览

```
# 第一卷 节奏概览

## 节奏分布
S5: ██ (2章) 6%
S4: ████████ (8章) 23%
S3: ████████████ (12章) 34%
S2: ████████ (8章) 23%
S1: ████ (5章) 14%

## 问题区域
- 第10-15章连续6章S2，节奏偏缓 ⚠️
- 第20-22章连续3章S4，可能疲劳 ⚠️
- 第25-30章节奏完美 ✅

## 追读力趋势
第1-5章：★★★★☆  (开篇吸引)
第6-10章：★★★☆☆  (节奏偏缓)
第11-20章：★★★★★  (高潮迭起)
第21-30章：★★★★☆  (完美收尾)

## 改进建议
- 第12章加入小高潮
- 第21章后安排缓冲
- 第28章增加Hook
```

## 四、使用指令

节奏分析已整合到 `/nw review` 指令中：

```
/nw review pacing 分析当前章节节奏
/nw review pacing volume 分析整卷节奏
/nw review pacing compare 对比前后章节节奏
/nw review pacing suggest 建议下一章节奏
/nw review pacing report 生成完整节奏报告
```

## 五、自动优化建议

### 检测到问题时的建议

| 问题 | 建议 |
|------|------|
| 连续3章S4+ | 下一章安排日常缓冲 |
| 连续3章S2- | 下一章安排冲突/高潮 |
| Hook超5章未回收 | 尽快兑现或说明原因 |
| 无爽点超过5章 | 安排打脸/逆袭/突破 |
| 信息密度过高 | 拆分到后续章节 |
| 信息密度过低 | 增加情节/设定/角色 |
| 情感单一 | 安排情感对比 |
| 角色出场不均 | 安排多角色互动 |
