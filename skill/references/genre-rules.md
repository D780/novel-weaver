# 题材规则大全

> 基于番茄小说、起点等平台热门小说总结的题材写作规则

## 番茄小说热门题材分类

### 一、男频热门题材

#### 1. 玄幻修仙（永恒爆款）

**代表作品：**
- 《开局地摊卖大力》- 弈青锋（番茄都市修仙爆款）
- 《斗破苍穹》- 天蚕土豆（退婚流鼻祖）
- 《凡人修仙传》- 忘语（凡人流开山）
- 《遮天》- 辰东（宏大世界观）

**核心要素：**
- 修炼体系清晰（等级划分、突破条件）
- 主角成长线明确（废柴→强者）
- 战斗场面精彩（招式、法宝、秘境）
- 爽点密集（越级挑战、打脸、奇遇）

**读者期待：**
- 升级快感、实力碾压
- 奇遇连连、机缘不断
- 热血战斗、快意恩仇

**禁忌：**
- 主角长期憋屈
- 战力体系崩坏
- 升级无逻辑
- 女主喧宾夺主

**爽点公式：**
```
开局废柴/被欺辱 → 获得金手指 → 逆袭打脸 → 收获资源 → 再次升级 → 循环
```

#### 2. 都市异能（贴近生活）

**代表作品：**
- 《大王饶命》- 会说话的肘子（负面情绪值系统）
- 《灵境行者》- 卖报小郎君（都市+诡秘）
- 《第一序列》- 会说话的肘子（废土都市）
- 《夜的命名术》- 会说话的肘子（赛博朋克）

**核心要素：**
- 现代都市背景
- 异能/系统/重生金手指
- 现实与奇幻结合
- 接地气的人物和场景

**读者期待：**
- 代入感强（贴近生活）
- 逆袭爽感（从平凡到卓越）
- 异能展示的酷炫
- 现实问题的解决

**禁忌：**
- 过于脱离现实
- 金手指过大失衡
- 都市感缺失

#### 3. 悬疑推理（高口碑）

**代表作品：**
- 《诡秘之主》- 爱潜水的乌贼（克苏鲁+悬疑）
- 《大奉打更人》- 卖报小郎君（仙侠探案）
- 《终局绝命》- 番茄悬疑神作（规则怪谈）

**核心要素：**
- 核心谜团贯穿全书
- 线索铺设和回收
- 推理逻辑严密
- 反转出人意料

**读者期待：**
- 烧脑解谜的快感
- 悬念持续的吸引力
- 真相大白时的震撼
- 智斗博弈的精彩

**禁忌：**
- 逻辑漏洞
- 机械降神
- 谜团自相矛盾

#### 4. 科幻末世（脑洞大开）

**代表作品：**
- 《吞噬星空》- 我吃西红柿（科幻修仙）
- 《深空彼岸》- 辰东（宇宙探索）
- 《长夜余火》- 爱潜水的乌贼（废土末世）

**核心要素：**
- 科学设定（可自洽）
- 末日/未来背景
- 生存压力
- 人性考验

**读者期待：**
- 想象力的满足
- 科技感的震撼
- 生存的刺激
- 人性的深度

**禁忌：**
- 伪科学硬伤
- 脱离科幻内核

#### 5. 历史穿越（知识流）

**代表作品：**
- 《回到明朝当王爷》- 月关（历史穿越巅峰）
- 《庆余年》- 猫腻（架空历史）
- 《步步生莲》- 月关（权谋历史）

**核心要素：**
- 历史背景考据
- 现代知识优势
- 权谋博弈
- 历史人物互动

**读者期待：**
- 历史代入感
- 现代降维打击
- 智谋博弈
- 改写历史的爽感

**禁忌：**
- 历史硬伤
- 过度现代化

#### 6. 系统流（爽文标配）

**核心要素：**
- 系统设定清晰
- 任务/奖励机制
- 升级路径明确
- 系统限制和代价

**读者期待：**
- 任务完成的成就感
- 奖励获取的爽感
- 系统升级的期待

**禁忌：**
- 系统功能混乱
- 奖励失衡
- 任务无意义

#### 7. 无限流（副本冒险）

**代表作品：**
- 《无限恐怖》- zhttty（无限流鼻祖）
- 《惊悚乐园》- 三天两觉（游戏无限流）

**核心要素：**
- 多副本/多世界
- 任务生存压力
- 团队协作/背叛
- 创意副本设计

**读者期待：**
- 新鲜感（每个副本不同）
- 紧张感（生死博弈）
- 智谋和勇气

**禁忌：**
- 副本同质化
- 主角光环过重

#### 8. 神豪流（消费爽文）

**核心要素：**
- 巨额财富来源
- 消费打脸情节
- 社会地位提升
- 人生赢家体验

**读者期待：**
- 花钱的快感
- 势利眼被打脸
- 人生逆袭

**禁忌：**
- 消费无意义
- 纯炫富无剧情

#### 9. 奶爸流（温情爽文）

**核心要素：**
- 主角是奶爸/带娃
- 萌娃互动
- 宠妻情节
- 家庭生活

**读者期待：**
- 萌娃的可爱
- 家庭温馨
- 宠妻甜度

#### 10. 洪荒流（神话体系）

**代表作品：**
- 《佛本是道》- 梦入神机（洪荒流鼻祖）

**核心要素：**
- 洪荒神话体系
- 圣人/大能设定
- 天道/因果/劫数
- 先天灵宝/功法

**禁忌：**
- 洪荒体系混乱

### 二、女频热门题材

#### 1. 言情甜宠（核心爆款）

**核心要素：**
- 男女主感情线
- 甜虐交替
- 霸总/暖男/忠犬男主
- 独立/可爱/坚韧女主

**读者期待：**
- 甜度爆表
- 情感共鸣
- 圆满结局

#### 2. 宫斗宅斗（权谋女性）

**核心要素：**
- 后宫/家族内斗
- 智谋博弈
- 地位争夺
- 女性成长

**读者期待：**
- 智斗的精彩
- 逆袭的快感
- 权谋的深度

#### 3. 穿越重生（逆袭改命）

**核心要素：**
- 时空穿越/重生
- 预知优势
- 弥补遗憾
- 改变命运

**读者期待：**
- 重生的爽感
- 逆袭的成功
- 遗憾的弥补

#### 4. 快穿（多世界恋爱）

**核心要素：**
- 多世界穿梭
- 不同男主类型
- 任务完成
- 节奏快

**读者期待：**
- 新鲜感（每个世界不同）
- 男主多样性
- 节奏紧凑

#### 5. 末世女强（生存成长）

**核心要素：**
- 末世背景
- 女主强大
- 空间/异能金手指
- 囤物资/建基地

**读者期待：**
- 女主的成长
- 囤货的满足感
- 生存的紧张感

### 三、中性热门题材

#### 1. 规则怪谈（新爆款）

**核心要素：**
- 规则设定
- 破解规则
- 诡异氛围
- 推理生存

**读者期待：**
- 恐怖的刺激
- 破解的成就感
- 规则的精妙

#### 2. 直播流（互动爽文）

**核心要素：**
- 直播设定
- 弹幕互动
- 观众反应
- 打赏/礼物

**读者期待：**
- 弹幕的搞笑
- 观众的震惊
- 打赏的爽感

#### 3. 反派流（视角创新）

**核心要素：**
- 主角是反派
- 对抗原主角
- 逆袭改命
- 道德灰色

**读者期待：**
- 视角的新鲜
- 反派的智慧
- 逆袭的成功

#### 4. 签到流（轻松爽文）

**核心要素：**
- 签到系统
- 每日奖励
- 轻松升级
- 无需努力

**读者期待：**
- 签到的期待
- 奖励的惊喜
- 轻松的阅读

## 题材选择建议

### 根据读者群体选择

| 读者类型 | 推荐题材 | 理由 |
|---------|---------|------|
| 男性18-25岁 | 玄幻修仙、都市异能、系统流 | 追求爽感和升级 |
| 男性25-35岁 | 悬疑推理、历史穿越、科幻 | 追求逻辑和深度 |
| 女性18-25岁 | 言情甜宠、快穿、宫斗 | 追求情感满足 |
| 女性25-35岁 | 重生逆袭、末世女强、宅斗 | 追求独立和成长 |
| 全年龄 | 规则怪谈、无限流、直播流 | 追求新鲜和刺激 |

### 根据写作难度选择

| 难度 | 推荐题材 | 说明 |
|------|---------|------|
| 入门 | 系统流、签到流、神豪流 | 套路成熟，容易上手 |
| 中等 | 玄幻修仙、都市异能、快穿 | 需要设定和规划 |
| 困难 | 悬疑推理、历史穿越、诡秘 | 需要逻辑和考据 |
| 专家 | 科幻末世、洪荒流、克苏鲁 | 需要知识和创意 |

## 题材融合趋势

### 当前热门融合

1. **都市+修仙**：现代都市背景下的修仙故事
2. **悬疑+仙侠**：仙侠世界中的探案推理
3. **科幻+修仙**：未来科技与修仙体系的碰撞
4. **系统+无限流**：系统加持下的多副本冒险
5. **直播+修仙**：修仙过程的直播互动
6. **反派+系统**：反派角色的系统逆袭
7. **规则怪谈+末世**：末世中的规则生存
8. **奶爸+修仙**：带娃修仙的温馨故事

### 融合建议

- **核心明确**：融合题材必须有主次
- **逻辑自洽**：不同元素必须融合自然
- **创新点突出**：融合必须带来新体验
- **读者接受度**：考虑目标读者的接受能力
