# AI味避免指南

## 什么是AI味

AI味是指AI生成文本中常见的、容易被识别的模式和特征。主要表现为：

1. **套话连篇** - 频繁使用固定句式
2. **词汇重复** - 同一词汇反复出现
3. **情感空洞** - 缺乏真实情感
4. **逻辑跳跃** - 转折生硬
5. **描写模板化** - 场景描写千篇一律

## AI常用套话清单

### 禁止使用的句式

| 套话类型 | 示例 | 替代方案 |
|---------|------|---------|
| 连续"了"字 | "他看了看，想了想，说了说" | "他端详片刻，斟酌着开口" |
| 连续"的"字 | "蓝色的天空的下面" | "蓝天下" |
| "不禁"开头 | "不禁感到" | 直接描写感受 |
| "不由得"开头 | "不由得想起" | 自然过渡到回忆 |
| "仿佛"滥用 | "仿佛...一般" | 具体描写 |
| "似乎"滥用 | "似乎...的样子" | 明确表达 |
| "一丝"滥用 | "一丝笑意" | 具体描写表情 |
| "一抹"滥用 | "一抹苦涩" | 直接描述情感 |

### AI高频词汇

以下词汇应谨慎使用，避免密集出现：

**情感类**
- 苦涩、复杂、微妙、难以言喻
- 心中涌起、内心深处
- 五味杂陈、百感交集

**动作类**
- 缓缓、默默、轻轻
- 深吸一口气、苦笑一声
- 皱了皱眉、点了点头

**描写类**
- 犹如、宛若、好似
- 映入眼帘、传入耳中
- 不知何时、不知不觉

## 避免AI味的技巧

### 1. 句式多样化

**错误示例（模板化）**
```
他缓缓抬起头，看着眼前的景象，心中涌起一阵复杂的情感。
```

**改进示例（多样化）**
```
他仰起脸。
眼前的画面让他喉咙发紧。
```

### 2. 具体化描写

**错误示例（空洞）**
```
她感到一阵难以言喻的悲伤。
```

**改进示例（具体）**
```
她眼眶发热，视线模糊了。手指无意识地绞着衣角，直到指节泛白。
```

### 3. 对话个性化

**错误示例（所有人说话一样）**
```
"此事颇为复杂，需从长计议。"
"确实如此，我们应当慎重行事。"
```

**改进示例（个性化）**
```
"这事儿没那么简单。"老王搓着手，"得慢慢捋。"
"嗯。"他点头，"不能急。"
```

### 4. 场景描写独特性

**错误示例（千篇一律）**
```
阳光透过窗户洒在地板上，形成斑驳的光影。
```

**改进示例（独特视角）**
```
午后的阳光斜切进屋子，在木地板上烙下一块金黄。灰尘在光柱里慢悠悠地转圈。
```

### 5. 情感表达层次化

**错误示例（直接告知）**
```
他感到很愤怒。
```

**改进示例（展示情感）**
```
他攥紧了拳头，指节泛白。牙关咬得咯咯响，喉结上下滚动了几下，终究还是没说出话来。
```

## 自检清单

写作完成后，检查以下项目：

- [ ] 是否连续使用了"了"字（超过2次）
- [ ] 是否连续使用了"的"字（超过2次）
- [ ] 是否频繁使用"不禁"、"不由得"
- [ ] 是否使用"一丝"、"一抹"等AI高频词
- [ ] 是否有连续3句以上使用相同句式
- [ ] 对话是否有个人特色
- [ ] 场景描写是否有独特视角
- [ ] 情感是否具体展示而非直接告知

## 反AI味检测规则

### 确定性规则（自动检测）

1. **连续"了"字**: 3个以上连续出现 → 警告
2. **连续"的"字**: 3个以上连续出现 → 警告
3. **套话密度**: 每千字超过5个AI高频词 → 警告
4. **句式重复**: 连续3句相同结构 → 警告
5. **词汇重复**: 同一形容词3次出现 → 建议

### 语义规则（AI辅助检测）

1. **情感空洞**: 检测是否只有情感告知而无具体描写
2. **对话同质化**: 检测不同角色对话是否风格相似
3. **描写模板化**: 检测场景描写是否使用常见模板
