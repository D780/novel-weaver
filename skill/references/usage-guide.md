# NovelWeaver 使用指南

## 快速开始

### 1. 初始化项目

```
/nw init 开始写一本玄幻修仙小说
```

AI 会引导你完成：
- 情绪标签选择（6大标签）
- 填写书名、题材、一句话简介
- 设置主角基本信息
- 选择文风（自动推荐）
- 确定目标平台和字数

### 2. 生成大纲

```
/nw plan 帮我生成总大纲
```

### 3. 开始写作（每次写后自动审查+字数检查+更新）

```
/nw write 写第一章，主角在精神病院醒来
```

### 4. 审查质量

```
/nw review 帮我看看这一章
```

## 进阶用法

### 灵感助手

```
/nw inspire                   # 获取灵感建议
/nw inspire 冲突设计          # 定向获取灵感
```

### 幕（Act）工作流

幕是卷内的剧情弧，有独立起承转合。当前幕章节写完后，使用 `/nw act`。

```
/nw act 下一幕怎么走
    ↓ 阶段一：AI展示创作全景（卷进度/角色/伏笔/情绪/字数）
    ↓ 阶段二：6条差异化分支 + AI推荐度 + 默认章节数
    ↓ 阶段三：用户选择 + 设定章节数（如"选B，写6章"）
        → AI分析偏离影响（🟢🟡🟠🔴）→ 提醒用户
        → 同步更新：大纲 + 卷计划 + 记忆 + 伏笔
        → 生成幕大纲（章节细纲）
    ↓ /nw write 进入写作循环
        每次写后：字数检查 + 更新大纲 + 更新记忆
```

**示例略览：**

```
用户：/nw act

AI：
## 📍 第一卷「青云宗的试炼」— 当前创作全景
- 卷进度：8/30章（2幕），主角击败内门弟子，长老欲收徒
- 待回收伏笔：隐藏血脉(🔴)、宗门令牌(🟡)
- 情绪位置：爆点刚释放，建议缓冲或回环

## 🎬 6条分支走向

```
A. [主线推进] 突破到筑基期，开始宗门大比剧情 (+4章)
B. [危机升级] 反派开始针对主角，主角遭遇暗杀 (+3章)
C. [支线展开] 探索秘境，结识新盟友 (+5章)
D. [缓冲沉淀] 日常修炼，加深与配角的互动 (+3章)
E. [回环收束] 收束前期伏笔，解决旧恩怨 (+3章)
F. [颠覆转向] 发现惊人真相，剧情方向大反转 (+4章)
```

用户：选B，写6章

AI：
## ⚠ 偏离分析：🟢 无偏离（本幕在总大纲计划内）
## ✅ 同步完成：幕计划+大纲+伏笔+记忆已更新
/nw write 写第9章
```

> 完整幕引导系统见 `references/act-guidance.md`

### 文风定制

```
日常对话使用弈青峰风格（搞笑接地气）
战斗场面使用天蚕土豆风格（热血爽快）
```

#### 分析个人风格

```
这是我的作品片段：[粘贴3-5段文字]
请分析我的写作风格，更新到 personal-voice.md
```

### 多线索管理

```
我有三条线索需要并行推进：
- 主线：主角升级
- 支线1：配角冒险
- 支线2：反派阴谋
请帮我规划如何交替推进
```

### 约束与规则

```
增加以下约束规则：
1. [规则1]
2. [规则2]
更新到创作宪法中
```

```
/nw review constraint           # 查看三轴混搭组合
/nw review constraint           # 检查是否使用常见套路
```

### 节奏与实体

```
/nw review pacing               # 分析本章节奏
/nw review pacing volume        # 查看整卷节奏曲线
/nw memory entity 角色           # 列出所有角色
/nw memory entity C001          # 查看指定实体
```

## 最佳实践

### 写作前

1. 完善创作宪法
2. 更新角色档案
3. 检查伏笔状态
4. 回顾前文摘要

### 写作中

1. 遵循大纲规划
2. 注意角色一致性
3. 设置章末悬念
4. 控制节奏快慢

### 写作后

1. 自动审查（AI自动完成）
2. 更新真相文件
3. 确认新实体
4. 选择后续走向

## 问题排查

| 问题 | 尝试 |
|------|------|
| 内容不符合预期 | 检查创作宪法、大纲是否完善；提供更具体的写作指引 |
| 前后矛盾 | `/nw review consistency`；检查真相文件是否更新 |
| 有AI味 | `/nw review AI味重不重`；参考 `references/anti-ai-patterns.md` |
| 节奏问题 | `/nw review pacing`；调整章节节奏评级 |
