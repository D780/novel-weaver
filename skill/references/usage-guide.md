# NovelWeaver 使用指南

## 快速开始

### 1. 初始化项目

```
/nw init 开始写一本玄幻修仙小说
```

AI 会引导你完成：
- 填写书名、题材、一句话简介
- 设置主角基本信息
- 选择文风（自动推荐）
- 确定目标平台和字数

### 2. 生成大纲

```
/nw plan 帮我生成总大纲
```

### 3. 开始写作

```
/nw write 写第一章，主角在精神病院醒来
```

### 4. 审查质量

```
/nw review 帮我看看这一章
```

### 5. 查看字数统计

```
/nw stats
```

### 6. 章节审查

```
/nw review 第一章
```

## 指令速查

> 指令分三级：**核心**（5个）→ **扩展**（6个）→ **高级**（子指令，极少用）

### 一级：核心指令

| 指令 | 说明 | 示例 |
|------|------|------|
| `/nw init` | 初始化项目 | `/nw init 开始写小说` |
| `/nw write` | 写作/续写 | `/nw write 写第一章` |
| `/nw review` | 审查（含一致性、AI味、节奏、约束） | `/nw review` |
| `/nw memory` | 查看/管理设定 | `/nw memory 角色都有谁` |
| `/nw help` | 帮助信息 | `/nw help` |

### 二级：扩展指令

| 指令 | 说明 | 示例 |
|------|------|------|
| `/nw plan` | 生成大纲 | `/nw plan 规划第一卷` |
| `/nw style` | 切换文风 | `/nw style 换辰东风格` |
| `/nw expand` | 扩写章节 | `/nw expand` |
| `/nw stats` | 字数统计 | `/nw stats` |
| `/nw inspire` | 获取灵感 | `/nw inspire` |
| `/nw summary` | 阶段总结 | `/nw summary` |

### 三级：高级指令（作为子指令）

| 子指令 | 归属 | 说明 |
|--------|------|------|
| `/nw review consistency` | `/nw review` | 一致性检查 |
| `/nw review pacing` | `/nw review` | 节奏分析 |
| `/nw review constraint` | `/nw review` | 约束检查 |
| `/nw review fix` | `/nw review` | AI 自动修复 |
| `/nw memory entity` | `/nw memory` | 实体管理 |
| `/nw analyze` | `/nw review` | 追读力分析 |

## 进阶用法

### 文风定制

#### 场景文风切换

```
日常对话使用弈青峰风格（搞笑接地气）
战斗场面使用天蚕土豆风格（热血爽快）
情感描写使用猫腻风格（细腻深刻）
世界观描写使用辰东风格（宏大深远）
```

#### 分析个人风格

```
这是我的作品片段：[粘贴3-5段文字]
请分析我的写作风格，更新到 personal-voice.md
```

### 多线索管理

```
我有三条线索需要并行推进：
- 主线：主角升级
- 支线1：配角冒险
- 支线2：反派阴谋
请帮我规划如何交替推进
```

### 约束与规则

#### 自定义约束

```
增加以下约束规则：
1. [规则1]
2. [规则2]
更新到创作宪法中
```

#### 查看三轴混搭

```
/nw review constraint
查看当前三轴混搭组合，确保章节不单调重复
```

#### 反套路检查

```
/nw review constraint
检查本章是否使用了常见套路，自动触发反套路改写
```

### 节奏管理

#### 章节节奏

```
/nw review pacing
分析当前章节节奏，确保张弛有度
```

#### 整卷节奏

```
/nw review pacing volume
查看整卷节奏曲线，避免连续紧张或平淡
```

### 实体管理

#### 实体提取

```
/nw memory entity 角色
列出所有角色实体及其状态
```

#### 实体详情

```
/nw memory entity C001
查看指定实体的详细信息
```

## 最佳实践

### 写作前

1. 完善创作宪法
2. 更新角色档案
3. 检查伏笔状态
4. 回顾前文摘要

### 写作中

1. 遵循大纲规划
2. 注意角色一致性
3. 设置章末悬念
4. 控制节奏快慢

### 写作后

1. 自动审查（AI自动完成）
2. 更新真相文件
3. 确认新实体
4. 选择后续走向

## 质量保障

### 自动审查

每次 `/nw write` 完成后，AI 会自动执行：

- **33维度审计**：情节推进、角色塑造、对话质量等
- **AI味检测**：套话统计、情感空洞、描述模式化
- **一致性检查**：角色OOC、设定冲突、时间线
- **追读力评分**：Hook质量、爽点分析、弃读风险

### 手动审查

```
/nw review 帮我看看这一章
/nw review consistency 前后有没有矛盾
/nw review analyze 这章好看吗
```

## 问题排查

### 内容不符合预期？

1. 检查创作宪法是否完善
2. 检查大纲是否清晰
3. 提供更具体的写作指引
4. 使用个人语料引导

### 前后矛盾？

1. `/nw review consistency 前后有没有矛盾`
2. 检查真相文件是否更新
3. 对比角色档案
4. 检查时间线

### 内容有AI味？

1. `/nw review AI味重不重`
2. 参考 `references/anti-ai-patterns.md`
3. 更新个人语料库
4. 提供更具体的写作指引

### 节奏问题？

1. `/nw review pacing 当前章节`
2. `/nw review pacing suggest`
3. 调整章节节奏评级

### 约束违反？

1. `/nw review constraint`
2. 检查三轴混搭组合
3. 确认约束继承

## 脚本工具

### 字数统计

```
/nw stats                   # 统计当前章字数
/nw stats 第三章            # 统计指定章
/nw stats volume            # 统计整卷字数
/nw stats all               # 统计全书字数
```

### 字数检查

```
/nw review consistency 字数是否达标  # 检查当前章字数
/nw review consistency 第三章字数    # 检查指定章字数
```

### 自动扩写

```
/nw expand                    # 扩写当前章至达标
/nw expand 增加500字          # 指定扩写量
/nw expand 扩写战斗场景       # 定向扩写特定场景
/nw expand 扩写对话部分       # 定向扩写特定部分
```

### 阶段总结

```
/nw summary                   # 查看总结列表
/nw summary 10chapters        # 生成小总结（每10章）
/nw summary 50chapters        # 生成大总结（每50章）
/nw summary volume            # 生成卷总结
/nw summary 查看 第10章总结   # 查看指定总结
```

### 灵感助手

```
/nw inspire                   # 获取灵感建议
/nw inspire 冲突设计          # 定向获取灵感
/nw inspire 我想写主角被困秘境 # 描述性灵感
/nw inspire 规划未来50章      # 长期规划建议
```

### 字数配置

在初始化时设置字数范围：
```
/nw init 写一本小说，每章2500-3500字
```

或修改创作宪法中的字数要求：
```
/nw memory 更新字数要求，每章3000-5000字
```
