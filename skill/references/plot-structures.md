# 情节结构模板

常见故事结构模板，用于规划小说章节。

## 三幕式结构

### 第一幕：设置（约25%）
| 章节 | 内容 |
|-----|------|
| 第1-2章 | 展现现状 + 激励事件 |
| 第3章 | 拒绝召唤后决定行动 |

### 第二幕：对抗（约50%）
| 章节 | 内容 |
|-----|------|
| 第4-5章 | 进入新世界，遭遇挑战 |
| 中点 | 重大转折/信息揭示 |
| 后续 | 困难升级，盟友可能背叛 |

### 第三幕：结局（约25%）
| 章节 | 内容 |
|-----|------|
| 最低点 | 一切看似失败 |
| 高潮 | 最终对决 |
| 终章 | 新常态 |

---

## 悬疑小说结构

| 章节 | 内容 | 钩子 |
|-----|------|------|
| 第1章 | 事件发生 | 谁干的？ |
| 第2-3章 | 接手+初步调查 | 线索指向谁？ |
| 第4-6章 | 审讯+新线索+第二事件 | 这是连环案？ |
| 第7-8章 | 陷入危险+反转 | 我们信任的人有问题？ |
| 终章 | 对决+解决 | 真的结束了吗？ |

---

## 言情小说结构

| 章节 | 内容 | 情感节点 |
|-----|------|---------|
| 第1-2章 | 分别介绍A和B | 读者共情 |
| 第3章 | 相遇（印象不佳） | 注定在一起但有冲突 |
| 第4-6章 | 被迫相处，吸引力增强 | 接近 |
| 第7-8章 | 误解/秘密/障碍 | 推远 |
| 第9章 | 关系破裂 | 看似无望 |
| 第10章 | 克服障碍 | 高潮 |
| 终章 | 在一起 | 圆满 |

---

## 惊悚/动作结构

```
第1章：危机出现 → 主角被追杀/威胁
第2章：应对计划 → 计划失败，情况恶化
第3章：追逐对抗 → 被逼入绝境
第4章：逆转机会 → 发现新希望，时间紧迫
第5章：最终对抗 → 生死一瞬
第6章：结局 → 后果
```

特点：节奏快、高密度事件、时间压力持续、悬念强度递增

---

## 反转结构

```
第1-2章：建立"真实"情况
第3-4章：出现疑点，不明显
第5-6章：第一次小反转
第7-8章：第二次反转
第9-10章：最终反转（一切颠覆）
```

关键：前期埋下看似无辜的线索，每次反转合乎逻辑

---

## 多线叙事结构

### 交叉剪辑模式
```
第1章：主角A故事
第2章：主角B故事
第3章：主角A推进
第4章：主角B推进
第5章：线索交汇
```

收敛原则：各线最终必须交汇，早期无关事件后来有关联

## 结构选择指南

| 故事类型 | 推荐结构 | 建议章节数 |
|---------|---------|-----------|
| 冒险/奇幻 | 三幕式/英雄之旅 | 8-12章/卷 |
| 侦探/悬疑 | 悬疑结构 | 8-10章 |
| 言情 | 言情结构 | 6-10章 |
| 动作/惊悚 | 惊悚结构 | 5-8章 |
| 心理/反转 | 反转结构 | 6-8章 |
| 多主角 | 多线叙事 | 10-15章 |
