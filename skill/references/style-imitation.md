# 文风模仿指南

> 如何在 NovelWeaver 中使用和自定义文风模仿

## 快速使用

### 1. 使用内置作者文风

在对话中直接告诉 AI：

```
请使用弈青峰的文风写作
```

或：

```
模仿辰东风格，写一段宏大战斗场景
```

### 2. 在创作宪法中指定

在 `.novel-weaver/memory/constitution.md` 中添加：

```markdown
## 文风模仿
- **主文风**：弈青峰（搞笑接地气）
- **战斗文风**：天蚕土豆（热血爽快）
- **情感文风**：猫腻（细腻深刻）
```

### 3. 混合多文风

```
日常对话用弈青峰风格（搞笑接地气）
战斗场面用天蚕土豆风格（热血爽快）
情感描写用猫腻风格（细腻深刻）
世界观描写用辰东风格（宏大深远）
```

## 自定义文风

### 方法一：提供作品片段

告诉 AI：

```
这是我的作品片段：[粘贴 3-5 段文字]

请分析我的写作风格，并生成文风配置文件
```

AI 会：
1. 分析句式特征
2. 提取常用词汇
3. 总结描写偏好
4. 生成配置文件到 `styles/authors/`

### 方法二：手动创建

创建 `styles/authors/我的文风.md`，参考以下模板：

```markdown
# [你的名字] 文风配置

## 作者信息
- **风格标签**：#[标签1] #[标签2] #[标签3]

## 核心写作原则
1. [原则1]
2. [原则2]
3. [原则3]

## 句式特征
### 高频句式
- [句式1]
- [句式2]
- [句式3]

## 描写规则
- **动作描写**：[特征]
- **场景描写**：[特征]
- **情感描写**：[特征]
- **对话风格**：[特征]

## 禁忌
- [禁忌1]
- [禁忌2]
- [禁忌3]
```

### 方法三：学习现有作者

告诉 AI：

```
我想学习弈青峰的搞笑风格，但加入自己的特色：
- 保持搞笑接地气
- 增加更多温情元素
- 减少网络用语

请帮我生成一个混合文风配置
```

## 文风切换

### 场景切换

```
【日常对话场景】使用弈青峰风格
- 轻松幽默
- 接地气
- 网络热梗

【战斗场景】使用天蚕土豆风格
- 热血激昂
- 节奏明快
- 招式帅气

【情感场景】使用猫腻风格
- 细腻深刻
- 不直白
- 有哲理

【宏大场景】使用辰东风格
- 气势磅礴
- 意境深远
- 世界观展开
```

### 章节切换

```
第1-3章：弈青峰风格（搞笑开局）
第4-10章：天蚕土豆风格（热血升级）
第11-20章：猫腻风格（情感深化）
第21-30章：辰东风格（宏大展开）
```

## 文风检查

写完一章后，告诉 AI：

```
请检查这章的文风是否符合弈青峰风格
```

AI 会检查：
- 句式是否符合特征
- 对话是否接地气
- 是否有搞笑元素
- 是否有AI味
- 是否符合描写规则

## 注意事项

### 保持一致性

- 同一章节不要频繁切换文风
- 一个场景用一个文风
- 跨章节切换要自然过渡

### 避免过度模仿

- 学习风格，不要照搬
- 保持自己的特色
- 不要过度使用特定作者的标志性句式

### 逐步融合

- 先用一个主文风
- 熟练后再混合
- 不要同时用太多文风
