# 唐家三少 文风配置

> 用于AI模仿唐家三少的写作风格

## 作者信息
- **代表作**：《斗罗大陆》《神印王座》《龙王传说》
- **题材**：玄幻、奇幻
- **风格标签**：#热血 #励志 #成长 #三观正 #团队

## 核心写作原则

### 1. 废柴逆袭
- 主角初始条件不好
- 靠努力坚持逆袭
- 成长路线清晰
- 每阶段有明确目标

### 2. 热血励志
- 主角坚韧不拔
- 永不言弃的精神
- 为了伙伴和目标奋斗
- 正能量满满

### 3. 团队作战
- 重视伙伴和团队
- 每个人都有自己的作用
- 团队配合精彩
- 不是主角一个人的秀

## 高频句式
```
"没有废物的武魂，只有废物的魂师"
"为了你，我愿意热爱整个世界"
"手握日月摘星辰，世间无我这般人"
"三十年河东，三十年河西，莫欺少年穷"
"我们是一个团队"
```

## 描写规则

### 动作描写
- 动作热血有气势
- 武魂/技能有特色
- 团队配合描写精彩

### 场景描写
- 场景宏大
- 学院、宗门、比赛场景
- 渲染热血氛围

### 情感描写
- 情感真挚感人
- 爱情、友情、亲情并重
- 情感推动成长

### 战斗描写
- 战斗有策略
- 武魂/技能配合
- 团队战精彩
- 战斗中突破成长

## 爽点设置

### 常用爽点
1. 团队配合胜利
2. 比赛逆袭
3. 武魂觉醒
4. 突破境界
5. 收获伙伴

### 爽点节奏
- 每3章一个小爽点
- 每10章一个大爽点
- 团队成长为主线

## 禁忌

- ❌ 不要主角不努力
- ❌ 不要团队没作用
- ❌ 不要力量体系混乱
- ❌ 不要主角太自私
- ❌ 不要过度使用AI套话
- ❌ 不要成长太突兀

## 开篇公式

```
主角觉醒 → 展示天赋/缺陷 → 进入学院 → 结识伙伴 → 团队成长
```

## 适用题材

- 玄幻修仙
- 热血励志
- 团队作战
- 学院流
- 成长逆袭
