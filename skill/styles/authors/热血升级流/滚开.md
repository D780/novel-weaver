# 滚开 文风配置

> 用于AI模仿滚开的写作风格

## 作者信息
- **代表作**：《极道天魔》《神秘之旅》《巫界术士》
- **题材**：玄幻、诡异、科幻
- **风格标签**：#黑暗 #杀伐果断 #加点升级 #极道 #硬核

## 核心写作原则

### 1. 黑暗杀伐
- 主角冷酷果断
- 不圣母，该杀就杀
- 世界残酷黑暗
- 力量就是一切

### 2. 加点升级
- 系统/金手指可以加点
- 数据化升级
- 快速提升实力
- 爽感直接

### 3. 极道流
- 主角追求极致力量
- 不断突破极限
- 肉身/功法修炼到极的
- 碾压一切敌人

## 高频句式
```
"深蓝，加点！"
"力量，才是真理"
"挡我者死"
"这一拳，你有几成把握接住？"
"极道之力，碾压一切"
```

## 描写规则

### 动作描写
- 动作简洁有力
- 一拳超人风格
- 力量感爆棚

### 场景描写
- 场景黑暗压抑
- 诡异恐怖氛围
- 渲染力量差距

### 情感描写
- 情感冷酷
- 不为感情拖累
- 目标明确

### 战斗描写
- 战斗碾压
- 力量感十足
- 一招秒杀
- 渲染恐怖实力

## 爽点设置

### 常用爽点
1. 加点升级
2. 实力碾压
3. 杀伐果断
4. 突破极限
5. 极道爆发

### 爽点节奏
- 每章都有升级
- 每3章一个小碾压
- 每10章一个大爆发

## 禁忌

- ❌ 不要主角心慈手软
- ❌ 不要战斗太拖沓
- ❌ 不要升级太慢
- ❌ 不要世界观太光明
- ❌ 不要过度使用AI套话
- ❌ 不要战力崩坏

## 开篇公式

```
主角穿越 → 获得金手指 → 疯狂加点 → 实力暴涨 → 碾压敌人
```

## 适用题材

- 极道流
- 黑暗玄幻
- 加点升级
- 杀伐果断
- 硬核爽文
