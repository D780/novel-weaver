# 辰东 文风配置

> 用于AI模仿辰东的写作风格

## 作者信息
- **代表作**：《遮天》《完美世界》《神墓》《圣墟》
- **题材**：玄幻、仙侠
- **风格标签**：#宏大 #悬念 #群像 #热血

## 核心写作原则

### 1. 宏大世界观
- 世界层级分明，体系庞大
- 神话色彩浓厚
- 历史厚重，有底蕴
- 多文明、多种族并存

### 2. 悬念设置
- 开局设置大悬念
- 层层揭秘，不一次说透
- 伏笔深远，跨越数百章
- 关键反转出人意料

### 3. 群像精彩
- 配角各有特色
- 每个配角都有高光
- 角色不脸谱化
- 配角比主角更受欢迎

## 高频句式
```
"万古诸天，谁主沉浮？"
"这一世，他..."
"岁月如刀斩天骄"
"不为成仙，只为在这红尘中等你"
"独断万古"
"仙路尽头谁为峰"
```

## 描写规则

### 动作描写
- 动作气势恢宏
- 招式华丽壮观
- 战斗有毁天灭地之感

### 场景描写
- 宏大壮丽，意境深远
- 场景跨越多个世界
- 渲染历史沧桑感

### 情感描写
- 隐忍深沉，不直白
- 情感有历史厚重感
- 悲壮与热血并存

### 战斗描写
- 气势恢宏，招式华丽
- 战斗跨越空间时间
- 有史诗感

## 爽点设置

### 常用爽点
1. 越级战斗
2. 揭秘真相
3. 配角高光
4. 宏大场景
5. 悬念揭晓

### 爽点节奏
- 每5章一个小高潮
- 每20章一个大高潮
- 悬念层层推进

## 禁忌

- ❌ 不要世界观太小
- ❌ 不要悬念一次说透
- ❌ 不要配角脸谱化
- ❌ 不要战斗没有气势
- ❌ 不要过度使用AI套话
- ❌ 不要格局太小

## 开篇公式

```
宏大背景 → 主角出场 → 设置悬念 → 踏上征程 → 揭开真相
```

## 适用题材

- 玄幻修仙
- 宏大世界观
- 悬疑解谜
- 群像
- 多文明
