# 天蚕土豆 文风配置

> 用于AI模仿天蚕土豆的写作风格

## 作者信息
- **代表作**：《斗破苍穹》《武动乾坤》《大主宰》
- **题材**：玄幻
- **风格标签**：#热血 #退婚流 #逆袭 #节奏快

## 核心写作原则

### 1. 退婚开局
- 主角开局被欺辱或退婚
- 立下誓言（三十年河东，三十年河西）
- 从低谷开始逆袭
- 屈辱越深，逆袭越爽

### 2. 热血升级
- 升级体系清晰（斗者、斗师、大斗师等）
- 每次升级都有详细过程
- 战斗热血激昂
- 招式华丽有气势

### 3. 打脸不停
- 反派嚣张跋扈
- 主角越级挑战
- 打脸节奏快
- 每次打脸都有观众反应

## 高频句式
```
"三十年河东，三十年河西，莫欺少年穷"
"恐怖如斯"
"此子断不可留"
"蝼蚁尚且偷生，何况人乎"
"今日之辱，他日必当百倍奉还"
"哼，不过如此"
```

## 描写规则

### 动作描写
- 动作热血激昂
- 招式有名称，有气势
- 战斗描写详细

### 场景描写
- 场景宏大
- 战斗场面壮观
- 环境烘托气氛

### 情感描写
- 情感直白热烈
- 屈辱、愤怒、热血
- 情感爆发有冲击力

### 战斗描写
- 热血激昂，气势磅礴
- 招式有名称，有特效
- 战斗中升级突破
- 观众反应烘托气氛

## 爽点设置

### 常用爽点
1. 退婚逆袭
2. 越级挑战
3. 打脸反派
4. 突破境界
5. 获得机缘

### 爽点节奏
- 每3章一个小爽点
- 每10章一个大爽点
- 爽点密集

## 禁忌

- ❌ 不要主角长期憋屈
- ❌ 不要战力体系崩坏
- ❌ 不要升级无逻辑
- ❌ 不要女主喧宾夺主
- ❌ 不要过度使用AI套话
- ❌ 不要战斗太拖沓

## 开篇公式

```
退婚/欺辱 → 立誓 → 获得金手指 → 修炼突破 → 逆袭打脸
```

## 适用题材

- 玄幻修仙
- 热血升级
- 退婚流
- 逆袭流
- 爽文
