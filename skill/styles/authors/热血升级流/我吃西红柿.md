# 我吃西红柿 文风配置

> 用于AI模仿我吃西红柿的写作风格

## 作者信息
- **代表作**：《星辰变》《盘龙》《吞噬星空》《莽荒纪》
- **题材**：玄幻、科幻
- **风格标签**：#升级 #宏大 #三观正 #工业化 #家庭

## 核心写作原则

### 1. 升级体系
- 等级明确清晰
- 地图不断扩展
- 升级过程详细
- 有成就感和期待感

### 2. 家庭情感
- 重视家人和伴侣
- 爱情线稳定
- 不后宫
- 家庭温馨感人

### 3. 三观端正
- 主角善良正直
- 有原则有底线
- 不滥杀无辜
- 正能量满满

## 高频句式
```
"我本微末凡尘，可也心向天空"
"一人得道，全家得道"
"修炼之路，步步艰难"
"浩瀚宇宙，强者为尊"
"为了保护家人，我必须变强"
```

## 描写规则

### 动作描写
- 动作简洁有力
- 升级过程详细
- 战斗有气势

### 场景描写
- 场景宏大
- 世界地图扩展
- 宇宙、星空、异界

### 情感描写
- 情感真挚
- 家庭温馨
- 爱情专一

### 战斗描写
- 战斗热血
- 招式清晰
- 升级突破

## 爽点设置

### 常用爽点
1. 升级突破
2. 地图扩展
3. 家庭团聚
4. 越级挑战
5. 获得宝物

### 爽点节奏
- 每5章一个小升级
- 每15章一个大升级
- 稳步成长

## 禁忌

- ❌ 不要升级无逻辑
- ❌ 不要主角花心
- ❌ 不要三观不正
- ❌ 不要地图太小
- ❌ 不要过度使用AI套话
- ❌ 不要烂尾

## 开篇公式

```
平凡少年 → 获得机缘 → 努力修炼 → 保护家人 → 走向世界
```

## 适用题材

- 玄幻升级
- 科幻星际
- 家庭温馨
- 三观正
- 工业化升级
