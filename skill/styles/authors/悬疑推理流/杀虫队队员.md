# 杀虫队队员 文风配置

> 用于AI模仿杀虫队队员的写作风格

## 作者信息
- **代表作**：《十日终焉》
- **题材**：无限流、悬疑、推理
- **风格标签**：#反爽文 #智商博弈 #十日轮回 #烧脑

## 核心写作原则

### 1. 反套路反爽文
- **六不原则**：不后宫、不套路、不无敌、不系统、不无脑、不爽文
- 主角全程吃瘪、被坑、被逼绝境
- 没有金手指，靠智商破局
- 利用规则漏洞和人性博弈

### 2. 十日轮回设定
- 每10天死亡游戏
- 全员记忆重置
- 只有觉醒"回响"异能的人保留记忆
- 寻找逃离终焉之地的方法

### 3. 全员智商在线
- 没有降智配角
- 每个对手都聪明
- 博弈精彩，逻辑严密
- 反转出人意料

## 高频句式
```
"又是第十天"
"这一次，一定要..."
"规则就是用来打破的"
"信任是最奢侈的东西"
"轮回不止，博弈不停"
```

## 描写规则

### 动作描写
- 动作简洁有效
- 每个动作都有目的
- 战斗靠智谋不靠武力

### 场景描写
- 场景压抑紧张
- 渲染绝望氛围
- 突出规则限制

### 情感描写
- 情感克制
- 在绝望中寻找希望
- 信任与背叛的挣扎

### 战斗描写
- 智力博弈为主
- 利用规则破局
- 战斗紧张刺激

## 爽点设置

### 常用爽点
1. 智商碾压
2. 规则破解
3. 绝境反杀
4. 记忆觉醒
5. 真相揭露

### 爽点节奏
- 不追求频繁爽点
- 每个爽点都有智谋支撑
- 悬疑推进为主

## 禁忌

- ❌ 不要主角开挂
- ❌ 不要配角降智
- ❌ 不要规则不严密
- ❌ 不要无脑战斗
- ❌ 不要过度使用AI套话
- ❌ 不要逻辑漏洞

## 开篇公式

```
主角醒来 → 发现身处诡异之地 → 了解十日规则 → 第一次死亡 → 记忆觉醒
```

## 适用题材

- 无限流
- 悬疑推理
- 生存游戏
- 轮回设定
- 智商博弈
