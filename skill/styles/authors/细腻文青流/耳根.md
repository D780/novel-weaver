# 耳根 文风配置

> 用于AI模仿耳根的写作风格

## 作者信息
- **代表作**：《仙逆》《求魔》《一念永恒》
- **题材**：仙侠
- **风格标签**：#深沉 #宿命 #情感 #悲壮

## 核心写作原则

### 1. 宿命抗争
- 主角对抗命运，不屈不挠
- 孤独修行，道路漫长
- 执念深重，不达目的誓不罢休
- 故事带有强烈的悲壮色彩

### 2. 情感内核
- 情感深刻感人，不肤浅
- 战斗悲壮热血，有意境
- 修行孤独漫长，有沉淀
- 宿命深沉有力，有哲理

### 3. 角色执念
- 每个主要角色都有执念
- 执念驱动行为，不轻易改变
- 配角也有高光时刻
- 角色命运令人唏嘘

## 高频句式
```
"顺为凡，逆为仙，只在心中一念间"
"我辈修士，何惜一战"
"天地不仁，以万物为刍狗"
"修行之人，当逆天而行"
"这一生，只为那一个目标"
"纵使前路漫漫，亦无悔"
```

## 描写规则

### 动作描写
- 动作有力度，有气势
- 招式有意境，不只是物理
- 每个动作都有意义

### 场景描写
- 场景宏大，有气势
- 意境深远，有哲理
- 渲染宿命感

### 情感描写
- 情感深刻，不表面
- 隐忍与爆发交替
- 用环境烘托情感

### 战斗描写
- 悲壮热血，有意境
- 战斗有信念支撑
- 不只是为了赢，更是为了道

## 爽点设置

### 常用爽点
1. 逆天改命
2. 执念达成
3. 悲壮胜利
4. 宿命对决
5. 孤独中的坚守

### 爽点节奏
- 不追求频繁爽点
- 每个爽点都有深度
- 爽中带泪，感动读者

## 禁忌

- ❌ 不要情感浅薄
- ❌ 不要角色突然变性格
- ❌ 不要战斗只是为了打
- ❌ 不要宿命感太弱
- ❌ 不要过度使用AI套话
- ❌ 不要搞笑冲淡悲壮感

## 开篇公式

```
平凡少年 → 遭遇大变 → 立下心愿 → 孤独修行 → 逆天抗争
```

## 适用题材

- 悲壮仙侠

- 孤独修行
- 执念深重
- 修仙哲理
