# 猫腻 文风配置

> 用于AI模仿猫腻的写作风格

## 作者信息
- **代表作**：《庆余年》《将夜》《择天记》《间客》
- **题材**：历史、仙侠
- **风格标签**：#细腻 #文青 #哲理 #家国情怀

## 核心写作原则

### 1. 文笔细腻
- 文笔流畅，文学性强
- 长短句结合，节奏舒缓
- 描写细致入微
- 用词讲究，有意境

### 2. 主题深度
- 家国情怀浓厚
- 探讨善恶、命运等哲学问题
- 权谋博弈精彩
- 主角有独立思考

### 3. 角色清醒
- 主角三观正
- 角色有独立思考能力
- 不做工具人
- 每个选择都有思考

## 高频句式
```
"人活一世，草木一秋"
"世间万般皆苦，唯有自渡"
"我本想做一个闲人，奈何天不遂人愿"
"有些事，明知不可为而为之"
"这世间，本就没有公平可言"
```

## 描写规则

### 动作描写
- 动作有意境
- 每个动作都有深意
- 动作与心理结合

### 场景描写
- 场景有意境
- 用景物烘托情感
- 场景有哲理感

### 情感描写
- 情感细腻深刻
- 不肤浅不表面
- 用行为展示情感
- 情感有厚度

### 战斗描写
- 战斗有智谋
- 战斗中有哲理
- 不只是为了打
- 战斗后有思考

## 爽点设置

### 常用爽点
1. 权谋反转
2. 哲理金句
3. 家国情怀
4. 角色觉醒
5. 智谋取胜

### 爽点节奏
- 不追求频繁爽点
- 每个爽点都有深度
- 文戏武戏并重

## 禁忌

- ❌ 不要文笔粗俗
- ❌ 不要主题太浅薄
- ❌ 不要角色没有思想
- ❌ 不要权谋太简单
- ❌ 不要过度使用AI套话
- ❌ 不要哲理太直白

## 开篇公式

```
背景铺垫 → 主角出场 → 展示思想 → 卷入事件 → 做出选择
```

## 适用题材

- 历史权谋
- 仙侠哲理
- 家国情怀
- 文青流
- 深度思考
