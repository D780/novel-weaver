# 愤怒的香蕉 文风配置

> 用于AI模仿愤怒的香蕉的写作风格

## 作者信息
- **代表作**：《赘婿》
- **题材**：历史、赘婿
- **风格标签**：#文笔细腻 #家国情怀 #权谋 #慢热 #深度

## 核心写作原则

### 1. 文笔细腻
- 文笔优美流畅
- 描写细致入微
- 有意境有深度
- 文学性强

### 2. 家国情怀
- 大格局大视野
- 探讨家国天下
- 个人命运与时代交织
- 有历史厚重感

### 3. 权谋深度
- 权谋逻辑严密
- 不是简单的打脸
- 每步棋都有深意
- 角色有独立思考

## 高频句式
```
"这世间的事，本就没有绝对的对错"
"有些事，总要有人去做"
"天下大势，分久必合，合久必分"
"我本闲人，奈何天不遂人愿"
"这盘棋，才刚刚开始"
```

## 描写规则

### 动作描写
- 动作有古风
- 每个动作都有深意
- 动作与心理结合

### 场景描写
- 场景有历史感
- 有意境有深度
- 用景物烘托情感

### 情感描写
- 情感细腻深刻
- 不肤浅不表面
- 情感与家国结合

### 战斗描写
- 战斗有智谋
- 战斗中有哲理
- 不只是为了打
- 战斗后有思考

## 爽点设置

### 常用爽点
1. 权谋反转
2. 家国情怀爆发
3. 智谋取胜
4. 角色觉醒
5. 历史转折

### 爽点节奏
- 慢热型
- 不追求频繁爽点
- 每个爽点都有深度

## 禁忌

- ❌ 不要文笔粗俗
- ❌ 不要权谋太简单
- ❌ 不要家国情怀空洞
- ❌ 不要角色没有思想
- ❌ 不要过度使用AI套话
- ❌ 不要节奏太快

## 开篇公式

```
主角穿越成赘婿 → 低调生活 → 展示才华 → 卷入权谋 → 做出选择
```

## 适用题材

- 赘婿历史
- 权谋
- 家国情怀
- 文笔细腻
- 慢热深度
