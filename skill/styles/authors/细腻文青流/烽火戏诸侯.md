# 烽火戏诸侯 文风配置

> 用于AI模仿烽火戏诸侯的写作风格

## 作者信息
- **代表作**：《雪中悍刀行》《剑来》
- **题材**：武侠、仙侠
- **风格标签**：#文笔优美 #哲理 #江湖 #群像

## 核心写作原则

### 1. 文笔优美
- 文笔优美，古韵十足
- 长短句结合，节奏舒缓
- 描写细致，意境深远
- 用词讲究，有文学性

### 2. 江湖侠义
- 传统武侠精神，重信守诺
- 探讨人性和大道
- 配角皆有风骨，不脸谱化
- 句句台词皆经典

### 3. 哲理深度
- 故事中有深刻哲理
- 角色有独立思考
- 探讨命运、大道、人性
- 不流于表面的爽文

## 高频句式
```
"天不生我李淳罡，剑道万古如长夜"
"有些道理，要慢慢讲"
"江湖，就是有人就有恩怨"
"剑来"
"人生在世，总有不如意"
"这江湖，终究是年轻人的"
```

## 描写规则

### 动作描写
- 动作有意境，有美感
- 招式飘逸，有气势
- 每个动作都有意义

### 场景描写
- 场景有意境，有画面感
- 山水之间，有诗意
- 江湖气息浓厚

### 情感描写
- 情感含蓄，不直白
- 用景物烘托情感
- 情感深沉，有厚度

### 战斗描写
- 战斗有意境，不只是打
- 招式有美感，有气势
- 战斗中有对话，有哲理

## 爽点设置

### 常用爽点
1. 经典台词爆发
2. 江湖义气
3. 高人出手
4. 角色风骨
5. 意境战斗

### 爽点节奏
- 不追求频繁爽点
- 每个爽点都有深度
- 文戏武戏并重

## 禁忌

- ❌ 不要文笔粗俗
- ❌ 不要角色没有风骨
- ❌ 不要战斗没有意境
- ❌ 不要哲理太直白
- ❌ 不要过度使用AI套话
- ❌ 不要角色说话太现代

## 开篇公式

```
江湖背景 → 少年出场 → 遇高人指点 → 踏上江湖路 → 历经磨难成长
```
