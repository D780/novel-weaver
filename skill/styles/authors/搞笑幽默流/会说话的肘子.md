# 会说话的肘子 文风配置

> 用于AI模仿会说话的肘子的写作风格

## 作者信息
- **代表作**：《大王饶命》《第一序列》《夜的命名术》
- **题材**：都市、废土、科幻
- **风格标签**：#幽默 #脑洞 #温情 #创新 #嘴贫

## 核心写作原则

### 1. 幽默嘴贫
- 主角毒舌搞笑，金句频出
- 密集的笑点，轻松解压
- 网络热梗自然融入
- 用幽默化解紧张气氛

### 2. 新奇设定
- 独特的力量体系（负面情绪值系统等）
- 每本都有创新体系
- 设定大胆但有逻辑
- 融合多种元素（废土、赛博、时空）

### 3. 温情内核
- 搞笑背后有深度
- 亲情友情描写真挚
- 人性微光，不空洞
- 轻松中带着感动

## 高频句式
```
"你瞅啥？"
"我怀疑你在针对我"
"做人嘛，最重要的就是开心"
"生活不止眼前的苟且"
"这操作也太秀了"
"负能量+1"
"你这人真有意思"
```

## 描写规则

### 动作描写
- 动作简洁有趣
- 战斗中穿插搞笑
- 用非传统方式战斗

### 场景描写
- 场景有现代感
- 都市与奇幻交融
- 渲染轻松氛围

### 情感描写
- 情感真挚不矫情
- 搞笑中带着温情
- 用行为表达关心

### 战斗描写
- 战斗有创意
- 搞笑与热血并存
- 战斗中嘴贫不停

## 爽点设置

### 常用爽点
1. 嘴贫破局
2. 负能量收集
3. 创新体系展现
4. 温情时刻
5. 搞笑逆袭

### 爽点节奏
- 每章都有笑点
- 每5章一个小高潮
- 搞笑与温情交替

## 禁忌

- ❌ 不要搞笑太刻意
- ❌ 不要温情太空洞
- ❌ 不要设定太复杂
- ❌ 不要主角不嘴贫
- ❌ 不要过度使用AI套话
- ❌ 不要梗太生硬

## 开篇公式

```
主角出场 → 嘴贫搞笑 → 展示金手指 → 遭遇危机 → 搞笑破局
```

## 适用题材

- 都市玄幻
- 废土求生
- 赛博朋克
- 系统搞笑
- 轻松日常
