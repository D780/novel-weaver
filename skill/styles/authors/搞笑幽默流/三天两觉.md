# 三天两觉 文风配置

> 用于AI模仿三天两觉的写作风格

## 作者信息
- **代表作**：《惊悚乐园》《贩罪》《鬼喊抓鬼》
- **题材**：无限流、悬疑、游戏
- **风格标签**：#吐槽 #玩梗 #反套路 #烧脑 #ACGN

## 核心写作原则

### 1. 吐槽玩梗
- 主角是吐槽鬼才，金句频出
- 大量ACGN梗，打破第四面墙
- 自嘲和自黑，不按套路出牌
- 用幽默化解紧张，用吐槽推进剧情

### 2. 反套路
- 不按常理出牌，主角是怪人
- 不按体系修炼，有自己的逻辑
- 超越性视角，看透一切套路
- 用非传统方式解决问题

### 3. 烧脑剧情
- 副本设计精巧，逻辑严密
- 反转出人意料，但合理
- 每个配角智商在线
- 智商博弈，不是武力碾压

## 高频句式
```
"欢迎来到人间游戏"
"我是写手"
"这不科学"
"根据我的经验..."
"这套路我熟"
"让我来给你分析一下"
"事情并没有你想的那么简单"
```

## 描写规则

### 动作描写
- 动作简洁但有创意
- 非传统战斗方式
- 用智慧和吐槽代替武力

### 场景描写
- 场景有游戏感
- 副本设计精巧
- 渲染诡异或荒诞氛围

### 情感描写
- 情感用吐槽包装
- 内心戏丰富
- 表面搞笑，内心深刻

### 战斗描写
- 战斗靠智谋和规则
- 非传统战斗方式
- 战斗中有大量吐槽

## 爽点设置

### 常用爽点
1. 神级吐槽
2. 反套路破局
3. 副本破解
4. 智商碾压
5. 打破第四面墙

### 爽点节奏
- 每章都有笑点
- 每3章一个反转
- 悬疑推进为主

## 禁忌

- ❌ 不要主角正经严肃
- ❌ 不要无脑战斗
- ❌ 不要配角降智
- ❌ 不要副本设计太简单
- ❌ 不要过度使用AI套话
- ❌ 不要梗太老旧

## 开篇公式

```
主角进入游戏 → 诡异场景 → 吐槽开场 → 发现规则 → 用非传统方式破局
```

## 适用题材

- 无限流
- 游戏文
- 悬疑推理
- 反套路
- 吐槽流
