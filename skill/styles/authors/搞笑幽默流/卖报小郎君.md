# 卖报小郎君 文风配置

> 用于AI模仿卖报小郎君的写作风格

## 作者信息
- **代表作**：《大奉打更人》
- **题材**：仙侠、探案、搞笑
- **风格标签**：#探案 #仙侠 #搞笑 #车速快 #群像

## 核心写作原则

### 1. 探案+仙侠
- 以案件推动剧情
- 每个案件都有悬念和反转
- 案件之间有关联
- 仙侠世界中的推理

### 2. 幽默搞笑
- 主角幽默风趣
- 车速快但不低俗
- 日常轻松有趣
- 用搞笑缓解紧张

### 3. 群像精彩
- 配角各有特色
- 女性角色鲜明
- 角色互动有趣
- 每个角色都有高光

## 高频句式
```
"此事必有蹊跷"
"打更人办事，闲杂人等回避"
"这案子不简单"
"且听我慢慢道来"
"这操作也太秀了"
```

## 描写规则

### 动作描写
- 动作简洁有趣
- 战斗中有搞笑元素
- 动作推动案件进展

### 场景描写
- 古代仙侠背景
- 场景有市井气息
- 渲染悬疑氛围

### 情感描写
- 情感轻松自然
- 感情线不拖沓
- 搞笑中带着真情

### 战斗描写
- 战斗有策略
- 战斗中有推理
- 战斗后有搞笑收尾

## 爽点设置

### 常用爽点
1. 破案反转
2. 幽默破局
3. 车速场景
4. 角色互动搞笑
5. 真相大白

### 爽点节奏
- 每章都有笑点
- 每5章一个案件高潮
- 悬疑与搞笑交替

## 禁忌

- ❌ 不要案件太简单
- ❌ 不要搞笑太低俗
- ❌ 不要推理有漏洞
- ❌ 不要角色脸谱化
- ❌ 不要过度使用AI套话
- ❌ 不要感情线太拖沓

## 开篇公式

```
命案发生 → 主角出场 → 调查线索 → 发现蹊跷 → 反转破案
```

## 适用题材

- 仙侠探案
- 搞笑修仙
- 悬疑推理
- 轻松日常
- 群像
