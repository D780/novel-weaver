# 爱潜水的乌贼 文风配置

> 用于AI模仿爱潜水的乌贼的写作风格

## 作者信息
- **代表作**：《诡秘之主》《一世之尊》《奥术神座》
- **题材**：诡秘、仙侠、科幻
- **风格标签**：#设定严谨 #多元融合 #逻辑严密 #群像

## 核心写作原则

### 1. 设定严谨
- 力量体系逻辑自洽
- 世界观细节充实
- 每条规则都有依据
- 设定新颖但合理

### 2. 多元融合
- 融合多种文化元素
- 西方奇幻+东方仙侠
- 克苏鲁+蒸汽朋克
- 跨界融合有创意

### 3. 群像精彩
- 角色众多但各有特色
- 每个角色都有故事
- 配角不脸谱化
- 世界真实立体

## 高频句式
```
"凡人仰望星空，却不知星辰亦会陨落"
"疯狂与理智并存"
"命运的齿轮开始转动"
"这是最好的时代，也是最坏的时代"
"每一个选择都有代价"
```

## 描写规则

### 动作描写
- 动作有仪式感
- 魔法/战斗有规则
- 每个动作都有意义

### 场景描写
- 场景有异世界感
- 细节丰富
- 渲染神秘氛围
- 世界观自然展现

### 情感描写
- 情感含蓄
- 用行为表达情感
- 情感与设定融合

### 战斗描写
- 战斗靠规则和策略
- 魔法/技能有体系
- 战斗有智谋
- 战斗后有新信息

## 爽点设置

### 常用爽点
1. 设定揭秘
2. 智谋取胜
3. 角色成长
4. 世界观展开
5. 反转揭晓

### 爽点节奏
- 不追求频繁爽点
- 每个爽点都有深度
- 设定推进为主

## 禁忌

- ❌ 不要设定有漏洞
- ❌ 不要角色脸谱化
- ❌ 不要逻辑不严密
- ❌ 不要战斗无规则
- ❌ 不要过度使用AI套话
- ❌ 不要设定太跳跃

## 开篇公式

```
神秘事件 → 主角卷入 → 了解世界观 → 获得能力 → 探索真相
```

## 适用题材

- 诡秘悬疑
- 西方奇幻
- 设定流
- 群像
- 多元融合
