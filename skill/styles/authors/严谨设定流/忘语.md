# 忘语 文风配置

> 用于AI模仿忘语的写作风格

## 作者信息
- **代表作**：《凡人修仙传》
- **题材**：仙侠
- **风格标签**：#严谨 #凡人流 #稳健 #资源至上

## 核心写作原则

### 1. 凡人修仙
- 主角资质平庸，无特殊天赋
- 靠努力和机缘一步步逆袭
- 修炼艰难，步步为营，不跳跃升级
- 资源才是硬道理，精打细算

### 2. 稳健谨慎
- 主角从不冒险，做事留有后手
- 低调行事，不张扬
- 生存第一，利益第二
- 遇到危险先想退路

### 3. 细节丰富
- 修炼过程详细描写
- 资源获取和使用精细计算
- 丹药、法宝、材料都有详细说明
- 战斗靠智慧和准备，不靠运气

## 高频句式
```
"修仙之路，步步艰难"
"资源才是硬道理"
"稳妥起见，还是..."
"韩老魔从不冒险"
"此物珍贵，不可浪费"
"先观察，再行动"
```

## 描写规则

### 动作描写
- 动作谨慎细致
- 每个动作都有目的
- 不轻易出手

### 场景描写
- 环境描写简洁
- 重点在资源分布
- 突出危险感

### 情感描写
- 情感内敛
- 不轻易表露真心
- 用行动表示关心

### 战斗描写
- 以弱胜强，靠智谋
- 战斗前充分准备
- 战斗中冷静分析
- 战斗后清点收获

## 爽点设置

### 常用爽点
1. 精心准备后反杀
2. 低调获得大机缘
3. 算计对手成功
4. 资源积累丰厚
5. 越级战斗靠智慧

### 爽点节奏
- 不追求密集爽点
- 每10章一个小高潮
- 稳步升级的成就感

## 禁忌

- ❌ 不要主角突然开挂
- ❌ 不要无脑越级挑战
- ❌ 不要主角性格大变
- ❌ 不要资源来源不明
- ❌ 不要战斗全靠运气
- ❌ 不要过度使用AI套话

## 开篇公式

```
平庸主角 → 获得机缘 → 谨慎修炼 → 低调成长 → 稳步逆袭
```

## 适用题材

- 凡人流修仙
- 稳健升级
- 资源争夺
- 低调逆袭
- 智谋型主角
