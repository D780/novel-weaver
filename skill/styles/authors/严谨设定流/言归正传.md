# 言归正传 文风配置

> 用于AI模仿言归正传的写作风格

## 作者信息
- **代表作**：《我师兄实在太稳健了》《天庭最后一个男神》
- **题材**：仙侠、搞笑
- **风格标签**：#稳健 #苟道 #反套路 #搞笑 #修仙

## 核心写作原则

### 1. 稳健苟道
- 主角极度谨慎
- 从不冒险，永远留后手
- 能躲就躲，不能躲就布局
- 安全第一，利益第二

### 2. 反套路
- 不按常理出牌
- 用非常规方式解决问题
- 主角脑回路清奇
- 用稳健打破一切套路

### 3. 搞笑日常
- 主角的稳健本身就是笑点
- 配角对主角的稳健感到无语
- 日常轻松有趣
- 搞笑中带着智慧

## 高频句式
```
"此事必有蹊跷，先观察三日"
"稳妥起见，还是..."
"我师兄实在太稳健了"
"能不出手就不出手"
"准备了十万张符箓，应该够了"
"此事，还需从长计议"
```

## 描写规则

### 动作描写
- 动作谨慎细致
- 每个动作都有备选方案
- 从不贸然出手

### 场景描写
- 场景有修仙氛围
- 渲染主角的谨慎
- 突出准备工作

### 情感描写
- 情感轻松自然
- 主角对稳健的执念
- 配角的无语和佩服

### 战斗描写
- 战斗靠准备和布局
- 十万张符箓砸人
- 战斗前有海量准备
- 战斗过程碾压

## 爽点设置

### 常用爽点
1. 海量准备碾压
2. 稳健破局
3. 反套路操作
4. 主角的谨慎搞笑
5. 布局成功

### 爽点节奏
- 每章都有笑点
- 每5章一个稳健高潮
- 搞笑与稳健交替

## 禁忌

- ❌ 不要主角不谨慎
- ❌ 不要准备不够充分
- ❌ 不要战斗太莽
- ❌ 不要搞笑太刻意
- ❌ 不要过度使用AI套话
- ❌ 不要稳健变懦弱

## 开篇公式

```
主角出场 → 展示稳健 → 遭遇事件 → 海量准备 → 稳妥解决
```

## 适用题材

- 稳健苟道
- 反套路
- 搞笑修仙
- 日常轻松
- 智谋型
