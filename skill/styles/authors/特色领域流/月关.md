# 月关 文风配置

> 用于AI模仿月关的写作风格

## 作者信息
- **代表作**：《回到明朝当王爷》《极品家丁》
- **题材**：历史、穿越
- **风格标签**：#历史穿越 #权谋 #后宫 #轻松

## 核心写作原则

### 1. 历史穿越
- 现代人穿越到古代
- 利用现代知识改变历史
- 历史细节丰富
- 古今碰撞有趣

### 2. 权谋博弈
- 朝堂斗争精彩
- 智谋对决
- 步步为营
- 权力游戏

### 3. 轻松幽默
- 主角风趣幽默
- 日常轻松有趣
- 后宫欢乐
- 用现代思维搞笑

## 高频句式
```
"这大明的江山，我来守护"
"山人自有妙计"
"且看我如何翻盘"
"古人诚不欺我"
"这一手，叫降维打击"
```

## 描写规则

### 动作描写
- 动作有古风
- 朝堂礼仪详细
- 战斗有谋略

### 场景描写
- 场景有历史感
- 古代城市、宫廷
- 渲染历史氛围

### 情感描写
- 情感轻松自然
- 后宫欢乐
- 古今思维碰撞

### 战斗描写
- 战斗靠谋略
- 以少胜多
- 智取为主

## 爽点设置

### 常用爽点
1. 现代知识碾压
2. 朝堂逆袭
3. 后宫欢乐
4. 智谋取胜
5. 改变历史

### 爽点节奏
- 每章都有笑点
- 每5章一个权谋高潮
- 轻松与权谋交替

## 禁忌

- ❌ 不要历史错误太多
- ❌ 不要后宫太混乱
- ❌ 不要权谋太简单
- ❌ 不要主角太现代
- ❌ 不要过度使用AI套话
- ❌ 不要古今脱节

## 开篇公式

```
穿越醒来 → 了解时代 → 展示现代知识 → 初次立功 → 进入权力中心
```

## 适用题材

- 历史穿越
- 权谋
- 轻松日常
- 后宫
- 古今碰撞

```
请使用月关的文风写作
```

或：
```
模仿月关风格，写一段...
```
