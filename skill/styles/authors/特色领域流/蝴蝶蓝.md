# 蝴蝶蓝 文风配置

> 用于AI模仿蝴蝶蓝的写作风格

## 作者信息
- **代表作**：《全职高手》
- **题材**：电竞、网游
- **风格标签**：#电竞 #群像 #热血 #幽默 #团队

## 核心写作原则

### 1. 电竞热血
- 游戏竞技精彩
- 操作描写详细
- 比赛紧张刺激
- 职业精神感人

### 2. 群像精彩
- 每个角色都有特色
- 队友各有专长
- 角色互动有趣
- 团队凝聚力强

### 3. 幽默日常
- 主角幽默风趣
- 日常轻松有趣
- 搞笑中带着热血
- 角色人设丰满

## 高频句式
```
"荣耀，从来不是一个人的游戏"
"休息一年，然后回来"
"你以为是终点，其实只是起点"
"战术，就是利用一切可利用的资源"
"我们是一个团队"
```

## 描写规则

### 动作描写
- 游戏操作详细
- APM、手速有描写
- 技能释放精准
- 战术配合精彩

### 场景描写
- 场景有电竞氛围
- 比赛现场紧张
- 训练室日常
- 渲染竞技氛围

### 情感描写
- 情感真挚
- 战友情深
- 对游戏的热爱
- 坚持与梦想

### 战斗描写
- 战斗即比赛
- 操作与战术并重
- 团队配合精彩
- 比赛有反转

## 爽点设置

### 常用爽点
1. 神级操作
2. 战术反转
3. 团队配合
4. 逆袭胜利
5. 角色成长

### 爽点节奏
- 每章都有笑点
- 每5章一场比赛
- 比赛有起伏

## 禁忌

- ❌ 不要游戏描写错误
- ❌ 不要团队没作用
- ❌ 不要比赛太简单
- ❌ 不要角色没有特色
- ❌ 不要过度使用AI套话
- ❌ 不要电竞味不浓

## 开篇公式

```
主角退役/低谷 → 重新开始 → 组建团队 → 训练提升 → 比赛逆袭
```

## 适用题材

- 电竞网游
- 团队竞技
- 游戏文
- 群像
- 热血励志
