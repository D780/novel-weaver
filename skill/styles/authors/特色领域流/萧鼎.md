# 萧鼎 文风配置

> 用于AI模仿萧鼎的写作风格

## 作者信息
- **代表作**：《诛仙》
- **题材**：仙侠
- **风格标签**：#古典仙侠 #情感深刻 #正邪之争 #虐心

## 核心写作原则

### 1. 古典仙侠
- 传统仙侠精神
- 正邪之辨
- 修道修心
- 意境深远

### 2. 情感深刻
- 爱情刻骨铭心
- 友情生死与共
- 师徒情深
- 情感推动剧情

### 3. 正邪之争
- 正邪界限模糊
- 好人也会做坏事
- 坏人也有苦衷
- 人性复杂

## 高频句式
```
"天地不仁，以万物为刍狗"
"何为正，何为邪？"
"这一生，只为一人"
"修道修心，心不正，道不存"
"情之所钟，正在我辈"
```

## 描写规则

### 动作描写
- 动作有仙气
- 剑招有意境
- 每个动作都有深意

### 场景描写
- 场景有仙侠意境
- 山水之间有灵气
- 渲染情感氛围

### 情感描写
- 情感深刻感人
- 虐心但美丽
- 情感与道义冲突

### 战斗描写
- 战斗有意境
- 剑招有诗意
- 战斗中有情感
- 战斗后有思考

## 爽点设置

### 常用爽点
1. 剑招意境
2. 情感爆发
3. 正邪反转
4. 角色成长
5. 生死抉择

### 爽点节奏
- 不追求频繁爽点
- 每个爽点都有情感深度
- 情感推进为主

## 禁忌

- ❌ 不要情感太肤浅
- ❌ 不要正邪太分明
- ❌ 不要战斗没有意境
- ❌ 不要角色没有成长
- ❌ 不要过度使用AI套话
- ❌ 不要仙侠味不浓

## 开篇公式

```
少年入仙门 → 结识挚友 → 遭遇情缘 → 正邪冲突 → 做出选择
```

## 适用题材

- 古典仙侠
- 情感深刻
- 正邪之争
- 虐心
- 修道修心
