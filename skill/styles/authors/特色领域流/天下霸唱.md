# 天下霸唱 文风配置

> 用于AI模仿天下霸唱的写作风格

## 作者信息
- **代表作**：《鬼吹灯》
- **题材**：盗墓、探险、悬疑
- **风格标签**：#江湖气 #民俗悬疑 #探险 #幽默

## 核心写作原则

### 1. 江湖气息
- 主角有江湖气
- 对话接地气
- 人物性格鲜明
- 市井智慧

### 2. 民俗悬疑
- 融入民间传说
- 风水、八卦、民俗
- 诡异但不恐怖
- 悬念层层推进

### 3. 探险解密
- 盗墓探险为主线
- 机关、陷阱、谜题
- 团队合作解密
- 每段探险都有收获

## 高频句式
```
"这地方邪门得很"
" según 老祖宗的说法..."
"不对劲，有古怪"
"咱们得小心点"
"这机关有点意思"
```

## 描写规则

### 动作描写
- 动作接地气
- 探险动作详细
- 机关破解过程

### 场景描写
- 场景诡异神秘
- 古墓、洞穴、秘境
- 渲染探险氛围
- 环境描写详细

### 情感描写
- 情感朴实
- 兄弟情义
- 幽默中带着紧张

### 战斗描写
- 战斗靠智慧和经验
- 与粽子/怪物的战斗
- 紧张刺激
- 战斗后有收获

## 爽点设置

### 常用爽点
1. 破解机关
2. 发现宝藏
3. 民俗知识应用
4. 团队配合
5. 揭开谜底

### 爽点节奏
- 每章都有新发现
- 每5章一个机关高潮
- 悬疑推进为主线

## 禁忌

- ❌ 不要机关太简单
- ❌ 不要民俗知识错误
- ❌ 不要角色没有江湖气
- ❌ 不要探险太顺利
- ❌ 不要过度使用AI套话
- ❌ 不要恐怖过度

## 开篇公式

```
神秘地图/线索 → 组建团队 → 进入古墓 → 机关陷阱 → 发现秘密
```

## 适用题材

- 盗墓探险
- 民俗悬疑
- 探险解密
- 民间传说
- 团队探险
