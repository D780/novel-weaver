# 三九音域 文风配置

> 用于AI模仿三九音域的写作风格

## 作者信息
- **代表作**：《我在精神病院学斩神》
- **题材**：都市、神话、热血
- **风格标签**：#多神话融合 #家国情怀 #守夜人 #热血

## 核心写作原则

### 1. 多神话体系融合
- 中国神话（西游、道教等）+ 希腊神话 + 北欧神话 + 日本神话 + 埃及神话
- 神明有人性，有欲望和执念
- 神话角色二次创造（如患PTSD的孙悟空）
- 宏大世界观，多体系并行

### 2. 灵魂角色串联
- 用一个个灵魂人物的故事串联主角剧情
- 配角比主角更受欢迎
- 每个配角有完整的人物弧光
- 配角有高光时刻，不是陪衬

### 3. 守夜人精神
- 守护与牺牲的主题
- 家国情怀浓厚
- 平凡人做不平凡的事
- 热血与感动并存

## 高频句式
```
"总有人要直面黑暗"
"守夜人，永不退缩"
"以凡人之躯，比肩神明"
"我们身后，就是万家灯火"
"神明也会流血"
```

## 描写规则

### 动作描写
- 动作热血有气势
- 战斗场面宏大
- 招式有神话色彩

### 场景描写
- 现代都市与神话交融
- 场景有视觉冲击
- 渲染守护氛围

### 情感描写
- 情感真挚感人
- 家国情怀浓厚
- 牺牲与守护的感动

### 战斗描写
- 热血战斗，有气势
- 神话能力展现
- 战斗中有情感爆发

## 爽点设置

### 常用爽点
1. 守夜人集结
2. 神明降临
3. 以凡人之躯对抗神明
4. 牺牲与守护
5. 家国情怀爆发

### 爽点节奏
- 每5章一个小高潮
- 每15章一个大高潮
- 情感与热血并重

## 禁忌

- ❌ 不要神明太冷漠
- ❌ 不要家国情怀空洞
- ❌ 不要角色没有弧光
- ❌ 不要战斗没有情感
- ❌ 不要过度使用AI套话
- ❌ 不要神话体系混乱

## 开篇公式

```
现代都市 → 神秘事件 → 主角觉醒 → 守夜人组织 → 对抗神明
```

## 适用题材

- 多神话体系
- 守夜组织
- 家国热血
- 都市神话
- 群像
